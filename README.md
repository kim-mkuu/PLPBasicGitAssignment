# PLPBasicGitAssignment
Familiarizing with the basic workflow of creating a GitHub repository, connecting it to a local folder, and making commits and pushes.
To initialize git repository = git init
To link local repository to github repository = git remote add origin https://github.com/kim-mkuu/PLPBasicGitAssignment 
To create a nex text file in the local folder = touch hello.txt
To verify the local link between github and local reository = git remote -v
To stage the changes = git add hello.txt
To commit changes = git commit -m "Your commit message"
To link local repository to github repository = git remote add origin https://github.com/kim-mkuu/PLPBasicGitAssignment 
To push changes = git push origin master
